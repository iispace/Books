 - 무릅 인대를 반사 망치로 두드리면 자동으로 다리가 올라가듯, 신체 신경 반사의 기저를 이루는 원칙이 감정 회로에도 똑같이 작동한다. 그래서 감정 회로가 자극을 받으면 우리는 반사적으로 공포, 분노나 적의로 반응한다. ....
   ...... 감정 회로를 촉발시키는 일이 반복되지 않는다면, 감정 회로는 화학 성분이 중성화되는 90초가 지나면 작동을 끝내고 멈출 것이다. p.22

- 우리는 생각을 하고, 감정을 느끼고, 생각과 감정에 대해 생리적으로 반응한다. p.30
