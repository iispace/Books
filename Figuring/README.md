# 진리의 발견(Figuring) - 마리아 포포바 지음, 도서출판 다름

- "지성을 잴 수 있는 단위는 없다." - Maria Mitchell

- "내 유일한 친구에게.<br>
 슬픔에 매여 움직이지 못하는 상태를 다시 한 번 깨뜨려 준 일에 그대에게 어떻게 감사를 표해야 할까요? 내 심장은 뛰고 있습니다. 나는 다시 한 번 살아가고 있습니다. 내가 그대에게 가치 있는 청중이라고, 내가 그대를 위해 존재할 이유가 충분하다고 느끼기 때문입니다.
 거장이여, 내 눈은 언제나 밝습니다. 나는 비록 가난하더라도 우주는 풍요롭다는 것을 봅니다. 나는 내 슬픔이 하찮다는 것을 봅니다. 내 의지 안에서 나는 포로가 아닙니다. 내 지성 안에서 나는 노예가 아닙니다...." - 베토벤 사후 16년에 베토벤에게 보낸 마가렛 풀러(Margaret Fuller)의 편지 중에서.
